# Seedling

Tree growth simulator and model generator.

## Getting Started

Easiest method is to check out the code and run:

```sh
npm install
npm start
```

## Uses:

* Electron
* React
* Three.js
* TypeScript
* Vite
* emotion
* Electron boilerplate copied from: https://github.com/caoxiemeihao/vite-react-electron
